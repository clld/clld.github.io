With people in Zurich and Nijmegen having installed the clld framework successfully (and on different platforms, no less), it was time to make a “real” Python package, i.e. publish clld on the Python Package Index. So here it is: clld on PyPI. This means that you can now install clld simply running 

pip install clld

This release emphasizes the third way to publish datasets with the help of the CLLD project. Thus, we took some steps to strip the software of any relics of our particular use case – publishing databases at clld.org. If there is demand we might go a step further, making portability to platforms other than our target platform a goal for the project.
